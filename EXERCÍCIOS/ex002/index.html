Parágrafos e quebras de linha

Para quebra de linha não adianta apertar 'enter' no teclado, pois o que o navegador entende são as tags.

A tag conhecida para quebra de linha é a tag <br>, mas também não é aconselhavél usa-la repetidas vezes quando se espera um resultado de "pular várias linhas", aprenderemos a fazer isso no CSS.
