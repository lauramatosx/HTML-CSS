Podemos mudar a forma que vai aparecer a "numeração" dessa lista, usando o atributo 'type' onde as opções são: 1 (numerica), A (alfabética maiuscula), a (alfabética mminuscula), I (numeros romanos maiusculo), i (numeros romanos minuscula), como por exemplo:

  1. Acordar
  2. Ligar para o João
  3. Escovar os dentes
  4. Lavar roupa

Ou:

  1. Acordar
  2. Ligar para o João
  3. Escovar os dentes
  4. Lavar roupa
