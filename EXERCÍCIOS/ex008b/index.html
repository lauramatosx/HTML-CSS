Código-fonte / Pré-formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

A tag CODE é ótima para destacar códigos, faz com o que a fonte fique mono espaçada, ou seja, todas os caracteres ocupam o mesmo espaço.

Mas essa tag não cria a formatação de um código, com quebra de linha e identação, para conseguir esse efeito é necessário usar a tag PRE antes da tag CODE.

        
            num = int(input('Digite um número'))
            if num % 2 = 0;
                print(f'O número {num} é PAR')
            else 
                print(f'O número {num} é IMPAR')

A tag PRE considera toda a formatação feita dentro dela e vai imprimir exatamente desse jeito.
